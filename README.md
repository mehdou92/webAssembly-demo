# DEMODAY - WebAssembly
#### Need :
- emsdk (https://github.com/emscripten-core/emsdk)
- Add to your bash emsdk path with: 
    - ``source ./emsdk_env.sh --build=Release``
    -   ``$ echo "source $(pwd)/emsdk_env.sh --build=Release > /dev/null" >> ~/.bashrc``
    
 #### Build project and start server
 - ``sh ./build.sh``
 - ``sh ./serve.sh``
